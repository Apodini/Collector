Usage

Regenerate the Protobuf files

To update and regenerate the generated gRPC and Probuf files you need to follow the following instructions:

  1. Install the Protocol Buffer Compiler, e.g. using brew install protobuf
  2. Install the Swift plugins by downloading the latest compiled version at https://github.com/grpc/grpc-swift/releases. Unpack the .zip and move the plugins to a location listed in your PATH variable. You might have to make them executable and run them manually first as they are not signed for macOS.
  3. Run sh generate.sh in the root of the repo.
